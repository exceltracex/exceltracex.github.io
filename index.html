Features

See the core features in action.

Each clip comes from a real Excel workbook. The examples show precedents, row consistency and the formula editor. Dependents and column consistency use the same UI and flow.

Trace precedents (and dependents) across sheets
Start from the cell you care about and drill down through its inputs across multiple sheets. The demo focuses on precedents, but the same interface can also trace dependents of a cell so you can see where a value is used.
  • List view and tree view for complex, multi-level formulas.
  • Start from one cell, then trace again from any traced cell to go deeper into the chain of precedents.
  • Formula bar highlight stays in sync with the selected node in the tracing window.
  • Level-based color highlighting improves formula readability as you explore.
Check row or column consistency in one pass
Scan a block of formulas and see which rows (or columns) follow the same pattern, and which ones are different. This example focuses on rows, but the same logic can be applied to columns when needed.
  • Detect rows or columns that accidentally use a different reference or offset.
  • Group consistent patterns together and highlight outliers that need attention.
  • Directly show the position of referenced cells so you always know what each part refers to.
Edit long formulas in a dedicated window
Instead of squeezing into the standard Excel formula bar, open the formula in an independent editor window with structure and color. You can keep the editor open while you move around the sheet to inspect other cells, without cancelling your edit.
  • Level-based color highlighting to improve readability of nested expressions.
  • Move the cursor in the formula and see the corresponding ranges highlighted in the sheet.
  • Step through tokens and sub-expressions to understand how the formula is built.
  • Because the editor window is independent, you don’t lose your editing state when you need to check other cells.
Additional features

Power tools for people who trace a lot.

These features don’t appear in the short demo clips, but they are designed to make real-world tracing and review work much faster.

History navigation and bookmarks
ExcelTraceX remembers where you have been while tracing and lets you mark important locations so you can return to them instantly.
  • Visit history with keyboard: use Ctrl + Alt + ← and Ctrl + Alt + → to move back and forward through previously visited cells and sheets, similar to browser back/forward.
  • Bookmarks: add bookmarks to important cells while tracing. Open the bookmark list to jump back to any bookmarked location without searching the workbook again.
  • These navigation tools work together with tracing and the formula editor, so you can explore, edit, and come back to key points smoothly.
